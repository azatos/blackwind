# BLACKWIND

## 1. 격주 문제 프로젝트
1. 비즈네르 암호


**코드 실행**

`wget https://github.com/azatos/blackwind/blob/master/1_vigenere.cpp`

`g++ -o vigenere 1_vigenere.cpp`

`./vigenere`
 

**옵션**
- 1 : Encrypt
  **input**
  - C : (string 입력)
  - K : (key 입력)
  **output**
  - M : (encrypted string)
- 2 : Decrypt
  **input**
  - M : (string 입력)
  - K : (key 입력)
  **output**
  - C : (decrypted string)
